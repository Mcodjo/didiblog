# 📈 Emploi Connect - Blog Professionnel d'Employabilité

## 🎯 Description du Projet

**Emploi Connect** est un blog professionnel spécialisé dans l'employabilité et la stratégie de carrière, développé par **Coach Didi (Cadnel DOSSOU)**. Le site s'adresse aux étudiants, jeunes diplômés et chercheurs d'emploi souhaitant décrocher un emploi plus rapidement.

### 🌟 Identité Visuelle
- **Logo officiel** : Logo Coach Didi intégré sur toutes les pages (header et footer)
- **Couleurs principales** : Orange (#f97316) et Bleu (#2563eb)  
- **Typography** : Inter (Google Fonts)
- **Ton de communication** : Personnel, bienveillant, expert (tutoiement)
- **Style** : Moderne, professionnel, accessible

## 📝 Notes de Développement

- Le site utilise **Tailwind CSS via CDN** (non recommandé en production)
- **Images via Unsplash** pour le développement (remplacer par assets réels)
- **JavaScript vanilla** sans framework pour simplicité et performance
- **Design mobile-first** avec breakpoints Tailwind standard
- **Accessibilité** respectée (ARIA labels, contraste, navigation clavier)

### 🔧 Pour Déployer en Production
1. Installer Tailwind CSS localement
2. Optimiser et compresser les assets
3. Configurer un service de newsletter
4. Ajouter le contenu réel des articles
5. Optimiser pour les moteurs de recherche
6. Configurer l'hébergement et le domaine
